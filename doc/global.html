lcWidth

Type:
  • Object
Properties:
Name Type Description
final calculatorCallback Calculates the final width of the node. The final width represents the outer width that the nodes element will assume when rendering the layout.
percent calculatorCallback Calculates the final width of the node in percent.
factor calculatorCallback Returns the proportional width factor of the nodes element, relative to the inner width of the parent nodes element (e.g.: 'half the width of the parent element' => factor = 0.5).
delta calculatorCallback Returns the difference of the outer and inner widths of the nodes element.
inner twoPlaceCalculatorCallback Substracts the outer margins and paddings of the nodes element from a given numerical value
outer twoPlaceCalculatorCallback Increases a given numerical value by the the outer margins and paddings of the nodes element.
